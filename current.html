Current Projects

Restaurant Website

I'm developing an example website for a fine dine dining restaurant using Visual Studio Code. It will be fully responsive and contain five main pages. The Home page will be a full-screen automated slideshow of photos showcasing the restaurant's interior. These images will also be in a controlled slideshow on the About page. The Menu page will be organized by tab buttons using jQuery UI. The reservation page will have a link to a page to make a reservation. This additional page will include a jQuery UI Datepicker. The Contact Us page will include a Google map API. I will design a logo for the restaurant in Krita.
